readyCallback :function

If a client starts asynchronously, you can register readyCallback, then the application will wait for the callback to ready

It will log when the callback is not invoked after 10s

Since:
  • 1.0.0
Source:

Example

```js
const done = app.readyCallback('mysql');
mysql.ready(done);
```
